Temperature Recording Client

This client has the task to record temperature measurement data on harddisk provided by temperature measurement clients through the MQTT protocol

The recording client shall get implemented in Python, the filename shall be mqtt_recording_client.py

In order to be able to run several instances of the recording client, the recording client shall be configurable through a configuration file.

The configuration file shall be formatted in yaml.

A sample configuration file is shown in tmcm_config_yml.txt

The configuration file shall be read by the recording client when it is started.

After reading the configuration file, the recording client shall publish the temperature sensing data according to the configuration file.

The sonsor identification mode (display individual physical sensor address) of the esp8266 SBC firmware does not need to get recording

The payload shall be structured in JSON as defined in payload_json.txt

The exact same MQTT functionality as in the esp8266 SBC firmware shall get provided

The sensordata published shall change with each message published as described in tmc_model_config_yml.txt

The recording client shall support graceful shutdown with ctrl+c, so that the client can be stopped without killing the process or truncating messages being sent.
